# ISO Documentation — MoneyMe

## Estándares ISO aplicados al desarrollo del proyecto

| ISO | Nombre | Propósito |
|-----|--------|-----------|
| **ISO/IEC 25010** | Software Quality Model | Evaluar y garantizar calidad del producto (funcionalidad, confiabilidad, usabilidad, rendimiento, seguridad, mantenibilidad, portabilidad) |
| **ISO/IEC 12207** | Software Life Cycle Processes | Organizar procesos de desarrollo, adquisición, configuración, operación y mantenimiento |
| **ISO/IEC 27001** | Information Security Management | Proteger datos de usuarios, autenticación JWT, cifrado, manejo seguro de credenciales |
| **ISO 9001** | Quality Management Systems | Asegurar mejora continua, documentación de procesos, enfoque al cliente, trazabilidad |

---

Cada subcarpeta/documento detalla cómo los principios de cada estándar se reflejan en la arquitectura, código, tests y documentación del proyecto.
